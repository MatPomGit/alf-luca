# luca-types

Wspólne modele danych i konfiguracji wykorzystywane w całym systemie LUCA.

## Public API

Sekcja odzwierciedla eksporty deklarowane w module inicjalizującym pakiet.

- `Detection`
- `TrackPoint`
- `CalibrationStatus`
- `InputConfig`
- `DetectorConfig`
- `TrackerConfig`
- `PostprocessConfig`
- `PoseConfig`
- `EvalConfig`
- `RunConfig`
- `run_config_from_entrypoint`
- `load_run_config`
- `save_run_config`
