# luca-reporting

Adapter wyjściowy do raportowania metryk i artefaktów analitycznych.

## Public API

- `compute_track_metrics`, `metrics_from_points`, `metrics_from_points_with_profile`
- `save_track_csv`, `save_all_tracks_csv`, `save_metrics_csv`, `save_track_report_pdf`
- `generate_trajectory_png`, `export_annotated_video`
- `compare_csv`, `load_tracking_csv`, `save_run_metadata`, `build_run_metadata`
