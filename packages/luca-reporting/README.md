# luca-reporting

Adapter wyjściowy do raportowania metryk i artefaktów analitycznych.

## Public API

Sekcja odzwierciedla eksporty deklarowane w module inicjalizującym pakiet.

- `build_run_metadata`
- `compare_csv`
- `compute_track_metrics`
- `generate_trajectory_png`
- `load_tracking_csv`
- `metrics_from_points`
- `metrics_from_points_with_profile`
- `save_all_tracks_csv`
- `save_metrics_csv`
- `save_run_metadata`
- `save_track_csv`
- `save_track_report_pdf`
- `export_annotated_video`
- `__version__`
